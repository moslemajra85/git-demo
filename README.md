# Git Demo

This is a small test project used for practicing how to create commits with the VS Code Git GUI.

The repository is intentionally simple. It currently contains a basic text file so changes can be made, staged, committed, and reviewed while learning the Git workflow in Visual Studio Code.

## Purpose

- Practice editing files in VS Code
- Practice staging changes with the VS Code Source Control panel
- Practice writing commit messages
- Practice creating commits through the VS Code GUI
